Welcome to the ui-extension-starter for IBM Blockchain Transparent Supply!

This application is an example of how to integrate into the Blockchain Transparent Supply user interface. Use this to learn:

  • How to subscribe to the user's token (See auth.service.ts)
  • How to use the token to call Blockchain Transparent Supply APIs (See data.service.ts)
  • How to communicate routing so that the URL in the browser changes as users navigate through your application (See frame-location-strategy.ts)
  • How to navigate to other UI modules (See product-details.component.ts)

The extension communicates with the parent UI through the use of postMessage. Open your browser's developer console to see the messages going back and forth.
